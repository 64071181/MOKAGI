# MOKAKI

# "updata":"202605171733"

自製最簡單的 ai agent

複製以下程式碼貼上主機執行即可

    curl -sL https://raw.githubusercontent.com/64071181/MOKAGI/refs/heads/main/MOKAGI.sh -o ~/MOKAGI.sh && sed -i 's/\r//' ~/MOKAGI.sh && bash ~/MOKAGI.sh

# 完整執行流程圖

    用戶消息
    │
    ├─ 0. 檢查 pending_confirm ──→ 是 → 執行命令 → 返回
    │        否
    │         ↓
    ├─ 0a. 檢查 pending_clarify ─→ 是 → 合併理解 → 確認/失敗 → 返回
    │        否
    │         ↓
    ├─ 1. 以 / 開頭？ ─────────────→ 是 → 直接命令 → 執行 → 返回
    │        否
    │         ↓
    ├─ 2. 意圖識別 (recognize_intent)
    │         │
    │         ├─ cmd == "chat" ──→ chat_with_tools → 記憶檢索 → call_llm → 返回
    │         ├─ 其他命令 ───────→ 執行工具 → 自然化 → 返回
    │         └─ cmd == None ────→ ask_clarification → 存入 _pending_clarify → 返回提問
    │
    └─ (多步工作流僅由 LLM 返回 /workflow 時觸發，經前端標記後調用 execute_multi_step)
